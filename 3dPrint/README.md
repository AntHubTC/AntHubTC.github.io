# 3D打印新手入门终极笔记

## 一、核心选择：设备、材料、配件
### （一）打印机选购
1. **新手首选类型**：FDM机型（操作简单、污染轻、后处理少，生态成熟），不推荐光固化（刺激性气味、后处理复杂，新手易翻车）。
2. **具体机型推荐（分预算/需求）**：
- 预算2000-3000元：拓竹A1combo（带AMS多色套装，打印稳、噪音小，国补后约2300元）、拓竹A1mini（小巧省空间，功能齐全）；
- 预算1000-2000元：Creality Ender-3系列（性价比之王，适合练手）；
- 需打大件/稳效果：创想三维K2（约3000元，封闭机身防变形，适配ABS/尼龙）；
- 兼顾稳定性：Stratasys F120（入门工业款，精度±0.2mm）。
3. **进阶方向参考**：未来想走专业路线，可关注Formlabs Form 3+（光固化，高精度手办/医疗模型）、LuxCreo Lux 3Li+（工业级DLP，大尺寸小批量生产）。

### （二）打印材料选择
1. **新手首选**：PLA材料（易打印、收缩小、不易翘边，刚性和细节在线），推荐平价款（避免溢价高的品牌），细分款按需选：
- 抗衰韧性：PLA+；
- 哑光质感：PLA CF（含碳纤维）、PLA mate；
- 丝绸光泽：silk材料；
- 特殊质感：PLA marble（石头感）、含木粉材料（木头感），还有夜光、耐高温、多色彩虹款。
2. **进阶材料（新手暂不尝试）**：PETG（易拉丝）、ABS（爱翘边）、软质TPU，熟悉基础操作后再解锁。
3. **材料使用注意**：料盘更换简单（扭开旧盘、对准新料、再拆绑带），切勿带料拆盘或提前拆绑带，避免材料散架。

### （三）必备配件
- 基础工具：水口钳（拆支撑）、胶水喷雾（防翘边，勿喷传动装置）、冷打板（提升PLA粘性，省电防翘边）；
- 辅助工具：400-800目砂纸（打磨PLA模型毛刺）、A4纸（辅助平台调平）。

## 二、完整打印流程+关键技巧（避坑版）
### （一）前期准备：找模型/建模
1. **找模型（新手优先）**：
- 平台推荐：拓竹make a word、创想云、国外T站、Thingiverse（免费多）、Cults（含优质付费）；
- 筛选技巧：优先选“无支撑、小尺寸”模型（如钥匙扣、手机支架），按“打印难度”筛选，无需先学建模。
- 模型格式：主流3MF格式，兼容多数切片软件（多基于bamboo studio）。
2. **建模（自定义需求可选）**：
- 软件选择：偏艺术（手办/摆件）选犀牛；偏工业（工具/零件）选Fusion 360、SOLIDWORKS；
- 关键要点：导出格式优化（犀牛导出STL调高三角面片数量；Fusion 360/SOLIDWORKS导出step格式），模型优化可打印性（少用支撑、增大底板接触面积）。

### （二）核心步骤：切片设置
1. 基础操作：模型拖入软件，大面积平面紧贴打印平台（提升稳定性），可缩放、上色、分割，多模型可分盘分批打印；
2. 关键参数（新手重点）：
- 材料匹配：选择对应材料类型（如PLA）；
- 工艺设置：打开“高级”，默认0.2mm标准配置，受力结构调整为3-4层墙、40%-50%填充，可选不同填充形状；
- 支撑添加：外伸角度大的面必须加支撑，优先选树桩支撑（拆件不伤模型），细高/薄壁零件可手动加支撑（防颤振）；
- 防倾倒设置：底面积小、重心不稳的模型（如细高花瓶），打开FM功能（外侧加平面，增强粘附）；
3. 最终检查：切片后预览，确认无漏支撑、方向正确。

### （三）打印操作：上机执行
1. 打印前必查：
- 平台调平（无自动调平功能需手动，喷嘴与平台间距约A4纸厚度）；
- 预热设置（PLA喷嘴190-210℃、热床50-60℃），避免空打、拉丝；
2. 文件传输：通过U盘或网络（手机APP远程）传至打印机，可远程查看打印状态；
3. 心态提示：新手可能遇到“炒面”“翘边”，属正常情况，总结经验即可。

### （四）后期处理：简单优化
- PLA模型：用400-800目砂纸逐步打磨毛刺，提升外观质感，无需复杂工具。

## 三、重要补充：环境、资源与认知
### （一）环境与安全
- FDM机型空气污染轻，避免放卧室；敏感体质或高频使用，搭配空气净化器。

### （二）学习资源与圈子
- 技术参考：Stratasys官网技术文档、Formlabs知识库；
- 交流平台：B站博主教程、3D打印贴吧、QQ群（实时解决翘边、堵头等突发问题）；
- 激励活动：B站“兴趣一键入坑指南”话题，分享内容有机会获千元奖励金。

### （三）技术边界认知
- 3D打印优势：小批量定制、复杂结构件制作（个性化收纳盒、手办）；
- 局限性：无法替代传统制造的大批量生产，理性看待打印效率与成本。
