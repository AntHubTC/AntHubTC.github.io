# 快捷键

## 快捷键表格

| 快捷键                     | 功能                         | 备注                   |
| -------------------------- | ---------------------------- | ---------------------- |
| **Ctrl + C**               | 取消当前输入/生成            | 通用中断               |
| **Ctrl + D**               | 退出整个会话                 | EOF                    |
| **Ctrl + L**               | 清屏                         | 保留历史               |
| **Ctrl + R**               | 终端命令历史搜索             | 需要终端支持           |
| **Shift + Tab**            | 切换模式（Plan/Auto/Normal） | 官方主要快捷键之一     |
| **Alt + M**                | 切换模式（部分平台）         | Windows 特别支持       |
| **Esc + Esc**              | Edit previous message/rewind | 不同于 slash `/rewind` |
| **↑ / ↓**                  | 上下历史                     | 浏览输入记录           |
| **Ctrl + V / Alt + V**     | 粘贴图片                     | 取决终端环境           |
| **\ + Enter**              | 多行换行                     | 所有终端               |
| **Option / Shift + Enter** | 多行换行                     | 需终端设置             |
