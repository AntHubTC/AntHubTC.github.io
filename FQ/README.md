# 财商

财商（FQ）全称**金融智商**，指个人或集体认识、创造及管理财富的能力，涵盖观念、知识、行为三个维度，由汤小明于2000年首次提出中文概念，并与智商、情商并称为现代社会三大素质。该术语源于罗伯特·清崎著作《富爸爸穷爸爸》，汤小明将其引入中国后推动财商教育普及，旨在帮助青少年树立正确的金钱观、价值观与人生观。



## 经商知识学习

> 经商不是学知识，是训练判断力。

## 一、先把一个误区掰正（很重要）

很多人一说“学经商”，第一反应是：

- 看商学院书
- 听老板演讲
- 报高价课程

但现实是：

> **你见过几个靠听课学会游泳的？**

经商能力的核心，不是“知道得多”，而是：

- 看懂局面
- 识别风险
- 做取舍
- 承担结果

这些东西，**只能在“真实决策”里长出来**。

------

## 二、我最推荐的 5 种学习方式（按性价比排序）

**① 用“老板视角”看世界（每天都能练）**

这是最便宜、最被低估的方式。

你可以刻意训练自己这样想：

- 这家店靠什么赚钱？
- 它的成本大头在哪？
- 如果生意不好，最先死在哪？
- 它最大的护城河是什么？

哪怕你只是：

- 吃饭
- 逛商场
- 刷电商

都在训练**商业直觉**。

👉 很多老板，真的就是靠“看多了”看明白的。

------

**② 小到不能再小的“真实生意”**

别一上来就谈创业。

你要的是：

> **可承担失败成本的商业试错**

比如：

- 帮人代买、代卖
- 二手转卖
- 本地服务撮合
- 小规模线上销售

哪怕只赚几百块，你学到的也包括：

- 客户怎么变卦
- 价格怎么被压
- 成本怎么被吃掉

这些东西，**书里没有**。

------

**③ 用“复盘”代替“鸡汤”**

这是区分普通人和未来商人的分水岭。

每一次尝试，强迫自己写三件事：

1. 我原来怎么判断的？
2. 实际发生了什么？
3. 下次我会改哪一条？

不管赚没赚钱，都写。

👉 **复盘能力 = 经商加速器**

------

**④ 只学“可落地”的商业模型**

别什么都学。

优先学这几类：

- 现金流生意（快进快出）
- 高复购模型
- 服务型生意
- 平台/中介型生意

原因很简单：

> **它们更接近普通人的现实。**

------

**⑤ 找“真在一线”的人，而不是“成功学IP”**

判断标准很简单：

- 他现在靠什么赚钱？
- 他说的东西能不能被反驳？
- 有没有失败经历？

**讲失败讲得清楚的人，通常更靠谱。**



## 《穷爸爸 vs 富爸爸》

### 核心差异对照表

| 维度         | 穷爸爸（典型中产思维）     | 富爸爸（资产思维）           |
| ------------ | -------------------------- | ---------------------------- |
| 对钱的态度   | 钱是安全感来源，越多越安心 | 钱是工具，用来放大能力       |
| 核心目标     | 稳定工作、固定收入         | 建立资产、持续现金流         |
| 对工作的看法 | 找一份“好工作”干一辈子     | 让钱替自己工作               |
| 收入来源     | 工资为主                   | 多元收入（资产、投资、企业） |
| 对风险的态度 | 害怕风险，尽量规避         | 管理风险，利用风险           |
| 对失败的看法 | 失败是羞耻，应尽量避免     | 失败是学习成本               |
| 学习重点     | 学好专业技能               | 学财商、法律、税务、投资     |
| 学校教育看法 | 学历决定未来               | 学历≠财务自由                |
| 对负债的认知 | 负债都是坏的               | 区分好负债和坏负债           |
| 对房子的看法 | 自住房是资产               | 自住房通常是负债             |
| 现金流思维   | 月月结余才安心             | 现金流是否持续更重要         |
| 对消费的态度 | 有钱了就改善生活           | 先买资产，再买享受           |
| 时间观念     | 用时间换钱                 | 用系统换时间                 |
| 税务认知     | 税后拿钱再消费             | 税前规划，合法降税           |
| 对规则的态度 | 遵守规则                   | 研究规则、利用规则           |
| 子女教育重点 | 考好学校、找好工作         | 财商教育、独立思考           |



### 一句话总结（非常重要）

- **穷爸爸：**

  > “我努力工作 → 赚钱 → 交税 → 消费 → 再继续工作”

- **富爸爸：**

  > “我学习规则 → 建立资产 → 产生现金流 → 让我有选择权”

> 对于技术人的你来说，要明白：**技术是杠杆，不是目的。**
