# Phaser

Phaser is a fast, free, and fun open source HTML5 game framework that offers WebGL and Canvas rendering across desktop and mobile web browsers and has been actively developed for over 10 years.



## 版本说明

### phaser3

目前是开源免费的。【所以学习选择了这个版本】

是官方在 2018 年发布的重大重构版本，完全推翻了 Phaser 2 的底层架构，针对现代浏览器和游戏开发需求进行了彻底优化。

目前是官方主推的版本，持续更新维护，社区活跃度最高。

官方开源仓库地址：[入口](https://github.com/phaserjs/phaser)

### phaser2

是 Phaser 的早期官方版本（发布于 2013 年前后），是框架的第一个成熟稳定版本，奠定了 Phaser 的核心功能（如物理引擎、渲染系统、资源加载等）。

官方在 2016 年后逐渐停止对其的主动开发，转向 Phaser 3 的研发。

### phaser CE

由于官方停止维护 Phaser 2，社区在 Phaser 2 的基础上 fork 并成立了「社区版」（CE），主要目的是修复 Bug、兼容新浏览器，并添加少量实用功能（不做大的架构改动）。

本质上是 Phaser 2 的「社区延续版」
